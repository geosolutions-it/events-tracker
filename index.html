Crime Data Dashboard

The crime data dashboard is a collection of infographics outlining data about criminal activities in the listed cities. The crime data includes the following;

  • Comparable Count of various crimes per precinct
  • Map displaying crime density
  • A line chart showing crime count over time
  • And a count of various crime by category
